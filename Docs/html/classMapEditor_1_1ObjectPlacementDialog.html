Detailed Description

Dialog for placing controllable objects on the map.

This dialog allows the user to place objects that can be controlled or interacted with on the map, such as barriers or elevators. It provides a form for specifying the object type, position, and associated switches if any.

MapEditor.ObjectPlacementDialog.addSwitchFields ( self)

Adds fields to the dialog for configuring a new switch.

Each switch can have its own type and position, which are added to the form as new inputs. This method dynamically updates the dialog to accommodate any number of switches.
